An open-source tool that combines LLM intelligence with static analysis to generate interactive knowledge graph dashboards for understanding codebases — reaching 1,000 GitHub stars within 5 days of launch.

- **Knowledge Graph**: Automatically extracts files, modules, functions, and their relationships into an interactive graph visualization.
- **Multi-Agent Pipeline**: Specialized agents (project-scanner, file-analyzer, architecture-analyzer, tour-builder) analyze codebases in parallel.
- **Interactive Dashboard**: Dark luxury themed React dashboard with graph-first layout, guided tours, and node-level deep dives.
- **Multi-Platform Support**: Works as a plugin for Claude Code, Cursor, Codex, OpenClaw, and OpenCode.
- **Understand Skills**: `/understand`, `/understand-chat`, `/understand-diff`, `/understand-explain`, `/understand-onboard` for different workflows.

Check out the [GitHub repository](https://github.com/Lum1104/Understand-Anything) for installation and usage.
